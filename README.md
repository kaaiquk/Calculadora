# Calculadora
Uma calculadora feita apenas em python com os meus conhecimentos atuais.

Conhecimentos usados:
- Loops While (especificamente o loop infinito While True)
- If & Else
- Operadores matemáticos

A ideia da calculadora foi utilizar o que eu sabia naquele momento em prática. O loop While faz com que o código se repita infinitamente, até que o usuário escolha a opção "SAIR", que cai em um break e finaliza o código. Essa repetição no código faz com que seja mais prático o uso, sem ter que executar várias vezes caso o usuário queira fazer multiplas operações.

Em meu ponto de vista, o código ainda é bem simples e fraco, acho que quanto mais eu for me aprofundando na programação vou encontrar formas de melhorar ele, como criar uma interface ou deixar o código mais limpo.
